### Search for github profiles using github graphql api

<img src="https://img.shields.io/badge/license-MIT-green" alt="MIT License"/>

## Technologies:

- React
- Apollo Client
- Github graphql API

## How to install:

Just clone this repo, install dependencies with yarn and npm and set up REACT_APP_GITHUB_GRAPHQL_TOKEN variable in .env.local file
